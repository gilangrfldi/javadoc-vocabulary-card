# Dokumentasi Proyek Java dengan JavaDoc
Proyek ini berisi kelas `VocabularyCard` yang merepresentasikan kartu kosakata bahasa Jepang dalam sistem pembelajaran berbasis permainan.

## Struktur Proyek
-   `src/` → Kode sumber Java
-   `docs/` → Dokumentasi HTML yang dihasilkan oleh JavaDoc

## Cara Mengakses Dokumentasi
1. Buka folder `docs/`
2. Klik file `index.html`
3. Tampilan dokumentasi akan muncul di browser

## Cara Menghasilkan Ulang JavaDoc
Jalankan perintah berikut di terminal (dari folder utama proyek):
```bash
javadoc -d docs src/VocabularyCard.java
```
